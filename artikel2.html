Fitur CSS

October 20, 2017 Views : 50

CSS1 fokus mengatur pemformatan dokumen HTML, seperti Font, Warna, teks, background, text attributes misalnya spasi antar baris, kata dan huruf, Posisi teks, gambar, table, margin border dan padding. CSS2 mengatur format dokumen untuk keutuhan cetak di printer, posisi konten, downloadable, font huruf, dan table layout. CSS3 mendukung tampilan desain website, animasi warna bahkan sampai animasi 3D, menunjang kompabilitas website dengan smartphone, CSS math, CSS obyek model, mendukungfungsi multimedia pada website, beberapa efek teks.
